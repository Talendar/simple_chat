## Simple chat
Simple implementation of a chat using sockets. Work in progress.


## Instructions

Install ncurses:

```
sudo apt install ncurses-dev
```

Compiling everything:

```
make all
```

Starting the server:

```
make server
```

To close the server just type:

```
exit
```
	
Starting a client:

```
make client
```

To close the application as a client just type:

```
\exit
```
